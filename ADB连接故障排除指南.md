# ADB连接故障排除 - 完整解决方案

## 📊 问题诊断

**当前状态：** ADB服务正常启动，但无法检测到设备

**可能原因：**
1. ✗ USB驱动未正确安装
2. ✗ USB线质量问题
3. ✗ 手机USB调试未正确配置
4. ✗ 端口冲突或防火墙阻止

---

## 🎯 解决方案（按优先级）

### 方案1：安装华为官方驱动（推荐）

**步骤：**

1. 下载华为HiSuite：
   - 官网：https://consumer.huawei.com/cn/support/hisuite/
   - 或搜索"华为HiSuite下载"

2. 安装HiSuite（会自动安装ADB驱动）

3. 安装完成后重启电脑

4. 连接手机，运行：
   ```batch
   cd 工具软件/platform-tools
   .\adb.exe devices
   ```

---

### 方案2：手动安装驱动

**步骤：**

1. 连接手机到电脑

2. 打开设备管理器：
   ```
   Win + X → 设备管理器
   ```

3. 查找带黄色感叹号的设备：
   - 可能名称：Android、Unknown Device、华为
   - 或在"其他设备"下

4. 右键 → 更新驱动程序

5. 选择：
   ```
   浏览计算机以查找驱动程序软件
   ```

6. 指定路径：
   ```
   工具软件\platform-tools
   ```

7. 完成安装

---

### 方案4：检查硬件

**测试清单：**

- [ ] 更换USB数据线（使用原装或高质量线）
- [ ] 更换USB端口（尝试USB 2.0，不用USB集线器）
- [ ] 尝试其他电脑
- [ ] 清洁手机USB接口
- [ ] 重启手机

---

### 方案5：手机端高级设置

**1. 开发者选项完整设置：**
```
设置 → 开发者选项
✓ USB调试
✓ 模拟点击位置
✓ 指针位置
✓ 显示触摸操作
```

**2. USB配置：**
```
设置 → 开发者选项 → USB配置
选择：MTP（媒体设备）
```

**3. 撤销并重新授权：**
```
设置 → 开发者选项 → 撤销USB调试授权
→ 拔掉USB线
→ 重新插入
→ 勾选"始终允许" → 点击允许
```

---

### 方案6：网络ADB（备选方案）

**前提：** 手机和电脑在同一WiFi网络

**步骤：**

1. 先通过USB连接一次（如果可能）

2. 启用网络ADB：
   ```batch
   .\adb.exe tcpip 5555
   ```

3. 获取手机IP：
   ```
   设置 → 关于手机 → 状态信息 → IP地址
   例如：192.168.1.100
   ```

4. 连接WiFi：
   ```batch
   .\adb.exe connect 192.168.1.100:5555
   ```

5. 验证：
   ```batch
   .\adb.exe devices
   ```

## 🔍 诊断命令

```batch
cd 工具软件/platform-tools
.\adb.exe version
```

### 检查端口占用
```batch
netstat -ano | findstr "5037"
```

### 重启ADB
```batch
.\adb.exe kill-server
.\adb.exe start-server
```
