## 八、故障排除

### 8.1 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| Termux:API 命令无法直接执行 | 权限限制 | 使用 AutoGLM 或手动操作 |
| Python 未安装 | 从未安装 | 运行 `pkg install python` |
| 传感器服务器不启动 | Python 缺失 | 先安装 Python |
| 权限被拒绝 | 非 root 设备 | 使用 ADB 或 AutoGLM |

### 8.2 调试命令

```bash
# 检查 Termux:API 是否安装
adb shell pm list packages | grep termux

# 检查 Termux 版本
adb shell dumpsys package com.termux | grep versionName

# 检查 Termux:API 权限
adb shell dumpsys package com.termux.api | grep permission

# 查看进程
adb shell ps | grep termux

# 查看日志
adb logcat | grep -E "(termux|sensor)"
```

## 十、快速命令参考

### 10.1 硬件访问命令

```bash
# ===== Termux:API 命令（需在 Termux 中执行）=====

# 传感器操作
termux-sensor -l                              # 列出所有传感器
termux-sensor -s Accelerometer -n 1          # 读取加速度计 1 次
termux-sensor -s Gyroscope -n 10             # 读取陀螺仪 10 次
termux-sensor -s Light -n 1                  # 读取光线传感器

# 语音合成
termux-tts-speak "Hello World"               # 朗读文本
termux-tts-engines                           # 列出 TTS 引擎

# 电池信息
termux-battery-status                         # 详细电池状态

# 摄像头（需要权限）
termux-camera-photo /sdcard/photo.jpg        # 拍照
termux-camera-info                           # 摄像头信息

# 录音（需要权限）
termux-microphone-record -f /sdcard/audio.wav -l 5  # 录音 5 秒

# ===== ADB 系统级命令（PC 端执行）=====

# 传感器信息
adb shell dumpsys sensorservice              # 完整传感器信息

# 电池信息
adb shell dumpsys battery                    # 电池状态

# 启动应用
adb shell am start -n com.termux/.app.TermuxActivity  # 打开 Termux
adb shell am start -a android.media.action.IMAGE_CAPTURE  # 打开相机

# 截图录屏
adb shell screencap -p /sdcard/screen.png    # 截图
adb shell screenrecord /sdcard/demo.mp4      # 录屏
```
