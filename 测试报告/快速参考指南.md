**最后更新:** 2026-03-07
**设备:** 荣耀 7 (Android 5.0.2)
**服务端口:** 9999

## 🚀 快速启动

### 启动 HTTP 服务器

```bash
# 方法 1: 使用启动脚本
cd /sdcard
./start_server.sh

# 方法 2: 手动启动
cd /sdcard
nohup python sensor_server.py > server.log 2>&1 &

# 验证服务器运行
curl http://127.0.0.1:9999/health
```

### PC 端连接

```bash
# 设置端口转发
adb forward tcp:9999 tcp:9999

# 测试连接
curl http://127.0.0.1:9999/health
```

## 🧩 可用传感器

| 传感器 | API 名称 | 说明 |
|--------|----------|------|
| 加速度计 | `accelerometer` | X/Y/Z 三轴加速度 |
| 陀螺仪 | `gyroscope` | 旋转角速度 |
| 光线 | `light` | 环境光照强度 |
| 气压 | `pressure` | 大气压力 |
| 磁场 | `magnetic` | 地磁场强度 |
| 温度 | `temperature` | 环境温度 |
| 距离 | `proximity` | 物体距离 |
| 重力 | `gravity` | 重力向量 |
| 线性加速度 | `linear_acceleration` | 不含重力的加速度 |
| 旋转向量 | `rotation_vector` | 设备方向 |
| 方向 | `orientation` | 罗盘方向 |

## 💻 Python 示例

```python
import requests
import time

BASE_URL = "http://127.0.0.1:9999"

# 获取传感器列表
sensors = requests.get(f"{BASE_URL}/sensors").json()
print(f"可用传感器: {sensors['count']} 个")

# 读取加速度计
accel = requests.get(f"{BASE_URL}/sensor/accelerometer").json()
x = accel['data']['Accelerometer']['x']
y = accel['data']['Accelerometer']['y']
z = accel['data']['Accelerometer']['z']
print(f"加速度: X={x:.2f}, Y={y:.2f}, Z={z:.2f}")

# 语音合成
requests.get(f"{BASE_URL}/tts", params={'text': 'Hello'})

# 电池状态
battery = requests.get(f"{BASE_URL}/battery").json()
print(f"电池: {battery['battery']['percentage']}%")
```

## 🌐 JavaScript 示例

```javascript
const BASE_URL = "http://127.0.0.1:9999";

// 获取传感器列表
fetch(`${BASE_URL}/sensors`)
  .then(r => r.json())
  .then(data => console.log(`传感器数量: ${data.count}`));

// 实时读取加速度计
setInterval(() => {
  fetch(`${BASE_URL}/sensor/accelerometer`)
    .then(r => r.json())
    .then(data => {
      const a = data.data.Accelerometer;
      console.log(`X: ${a.x}, Y: ${a.y}, Z: ${a.z}`);
    });
}, 1000);  // 每秒读取

// 语音合成
fetch(`${BASE_URL}/tts?text=你好`);
```

## 🔧 故障排除

### 服务器无法启动
```bash
# 检查端口占用
netstat -an | grep 9999

# 停止现有进程
killall python

# 查看错误日志
cat /sdcard/server.log
```

### API 无响应
```bash
# 检查服务器状态
ps | grep python

# 测试本地连接 (在手机 Termux 中)
curl http://127.0.0.1:9999/health

# 重启服务器
killall python && cd /sdcard && ./start_server.sh
```

### ADB 断开
```bash
# 重启 ADB
adb kill-server && adb start-server

# 检查设备连接
adb devices

# 重新设置端口转发
adb forward tcp:9999 tcp:9999
```

### 传感器报错
```bash
# 测试 termux-sensor 命令
termux-sensor -l

# 检查 Termux:API 权限
# 设置 → 应用 → Termux:API → 权限
```

## 🔐 安全提示

1. **不要暴露到公网** - 当前服务无认证机制
2. **使用防火墙** - 限制访问来源
3. **添加 API 密钥** - 生产环境建议实现认证
4. **HTTPS** - 敏感数据使用加密传输
5. **速率限制** - 防止 API 滥用
