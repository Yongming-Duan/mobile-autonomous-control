## 问题记录

### 问题1：ADB版本不匹配

**现象：**
```
adb server version (39) doesn't match this client (41)
```

**影响：** 警告信息，不影响功能

**解决方案：**
- 统一使用platform-tools中的ADB
- 创建 `adb_stable_manager.bat`

**状态：** ✅ 已解决

---

### 问题2：AutoGLM运行时连接断开

**现象：** 切换到AutoGLM时ADB断连

**根本原因：**
- 多个ADB进程冲突
- 连接参数未优化

**解决方案：**
- 创建 `autoglm_stable.bat`
- 创建 `autoglm_one_click.bat`
- 优化连接参数

**状态：** ✅ 已解决

---

### 问题3：dumpsys window无输出

**现象：**
```
ValueError: No output from dumpsys window
```

**原因：** Android 5.0.2系统较旧

**影响：** 轻微，不影响核心功能

**状态：** ⚠️ 已知限制
